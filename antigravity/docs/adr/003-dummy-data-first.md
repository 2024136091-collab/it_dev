# ADR-003: 더미 데이터 우선 개발

## 상태
채택됨 (API 연동은 13주차 예정)

## 배경
국가승강기정보센터 API는 공공 API 인증키 발급과 CORS 설정이 필요해 초기부터 연동하면 개발 속도가 느려진다.

## 결정
`ApiRepository`에 국가승강기정보센터 API 구조와 동일한 더미 데이터 16건을 하드코딩하고, 실제 API 연동은 이후 단계에서 진행한다.

## 근거
- API 인증키 없이도 전체 기능 개발·테스트가 가능하다.
- 더미 데이터 ID를 실제 국승정 번호 형식(`XXXX-XXX`)으로 맞춰 향후 교체가 쉽다.
- `ElevatorRepository` 인터페이스를 통해 `ApiRepository`를 교체해도 상위 레이어는 변경 없다.

## 더미 데이터 구성

| 건물 | 지역 | 유형 |
|---|---|---|
| 신구대학교 본관·도서관 | 경기·인천 | 학교 |
| 강남 파이낸스센터 | 서울 | 오피스 |
| 롯데월드타워 | 서울 | 상업시설 |
| 판교 알파돔타워 | 경기·인천 | 오피스 |
| 휴먼시아아파트 | 경기·인천 | 아파트 |
| 분당 파크뷰 아파트 | 경기·인천 | 아파트 |
| 서울아산병원 | 서울 | 병원 |

## 결과
- 실제 API 연동 시 `ApiRepository.search()·getAll()`만 수정하면 된다.
- 나머지 레이어(Domain·Application·Presentation)는 변경 불필요하다.
